영화 온라인 예매

사용자가 영화 골라 -> User, Movie
영화 골랐으면 상영관에서 자리 골라 -> Theater, Seat
결제해

영화에는 감독이랑 배우 있어 -> Director, Actor
감독이 여러 영화를 찍을 수 있고, 배우는 여러 영화에 출연할 수 있어
하나의 영화에 여러 배우가 출연할 수는 있지만, 여러 감독이 하나의 영화를 찍을 수 있어

영화관에는 영화마다 상영 시간이 있어(시작 시간, 종료 시간, 상영관 이름)
상영관마다 여러 좌석 있고, 상영 시간마다 여러 개 좌석 있어
좌석마다 여러 상영 시간을 가질 수 있지

사용자의 예매 내역을 조회할 수 있어야 하니까 사용자는 여러 개의 예매를 갖겠네
여러 개 좌석 예매할 수 있으니까 예매마다 여러 개 좌석 있을 수 있겠네
근데 또 예매 하나 당 상영 시간은 하나만 선택할 수 있네
